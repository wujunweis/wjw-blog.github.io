React Hook

hooks жҰӮеҝө

  • Hook жҳҜ React 16.8 зҡ„ж–°зү№жҖ§пјҢе®ғеҸҜд»Ҙи®©дҪ еңЁдёҚзј–еҶҷclass зҡ„жғ…еҶөдёӢдҪҝз”Ё State д»ҘеҸҠе…¶д»–зҡ„ React зү№жҖ§

    д»ҺиҝҷеҸҘиҜқеҸҜд»Ҙеҫ—зҹҘ:Hook еўһеҠ дәҶеҮҪж•°ејҸз»„д»¶дёӯзҡ„Stateзҡ„дҪҝз”ЁпјҢ еңЁеҮҪж•°ејҸз»„д»¶жҳҜж— жі•жӢҘжңүиҮӘе·ұзҡ„зҠ¶жҖҒзҡ„пјҢеҸӘиғҪйҖҡиҝҮProps д»ҘеҸҠ Content жқҘжёІжҹ“иҮӘе·ұзҡ„UIпјҢ иҖҢеңЁдёҡеҠЎйҖ»иҫ‘дёӯжңүдәӣеңәжҷҜеҝ…йЎ»дҪҝз”ЁеҲ°StateпјҢ йӮЈд№ҲжҲ‘д»¬еҸӘиғҪе°ҶеҮҪж•°ејҸз»„д»¶е®ҡд№үдёәClass з»„д»¶гҖӮ йӮЈд№Ҳд»ҺзҺ°еңЁејҖе§ӢпјҢ еҲҷдёҚйңҖиҰҒж”№дёәClassз»„д»¶дәҶгҖӮ

  • React 16.8 еҠ е…Ҙ hooks пјҢи®©ReactеҮҪж•°ејҸз»„д»¶жӣҙеҠ зҒөжҙ»
  • hooks еңЁд№ӢеүҚReactзҡ„зүҲжң¬еӯҳеңЁеҫҲеӨҡй—®йўҳ
    1. еңЁз»„д»¶й—ҙеӨҚз”ЁзҠ¶жҖҒйҖ»иҫ‘пјҲеҫҲйҡҫеҒҡеҲ°пјү
    2. еӨҚжқӮз»„д»¶еҸҳеҫ—йҡҫд»ҘзҗҶи§ЈпјҢй«ҳйҳ¶з»„д»¶еҠ еҮҪж•°ејҸз»„д»¶зҡ„еөҢеҘ—иҝҮж·ұгҖӮ
    3. class з»„д»¶зҡ„thisжҢҮеҗ‘й—®йўҳпјҲеөҢеҘ—еӨҡдәҶпјҢе…ізі»еӨҚжқӮпјү
    4. йҡҫд»Ҙи®°еҝҶзҡ„з”ҹе‘Ҫе‘Ёжңҹ
  • hooks еҫҲеҘҪзҡ„и§ЈеҶідәҶдёҠиҝ°зҡ„й—®йўҳпјҢhooksжҸҗдҫӣеҫҲеӨҡж–№жі•еҰӮпјҡ
    1. useState иҝ”еӣһжңүзҠ¶жҖҒеҖјд»ҘеҸҠжӣҙж–°иҝҷдёӘзҠ¶жҖҒеҖјзҡ„еҮҪж•°
    2. useEffect жҺҘеҸ—еҢ…еҗ«е‘Ҫд»ӨејҸпјҢжңүеүҜдҪңз”Ёд»Јз Ғзҡ„еҮҪж•°
    3. useContent жҺҘеҸ—дёҠдёӢж–ҮеҜ№иұЎпјҲд»ҺReact.createContext зҡ„иҝ”еӣһеҖјпјүе№¶иҝ”еӣһеҪ“еүҚдёҠдёӢж–Үзҡ„еҖј
    4. useReducer useState зҡ„жӣҝд»Јж–№жЎҲпјҢжҺҘеҸ—зұ»еһӢдёәпјҲstateпјҢactionпјү=> newState зҡ„reducer е№¶иҝ”еӣһдёҺdispatchж–№жі•еҢ№й…Қзҡ„еҪ“еүҚзҠ¶жҖҒ
    5. useCallBack иҝ”еӣһдёҖдёӘеӣһеҝҶзҡ„memoizedзүҲжң¬пјҢиҜҘзүҲжң¬д»…еңЁе…¶дёӯдёҖдёӘиҫ“е…ҘеҸ‘з”ҹж”№еҸҳж—¶жүҚдјҡеҸҳжӣҙпјҢзәҜеҮҪж•°зҡ„иҫ“е…Ҙиҫ“еҮәзЎ®е®ҡжҖ§
    6. useMemo зәҜзҡ„дёҖдёӘи®°еҝҶеҮҪж•°
    7. useRef иҝ”еӣһдёҖдёӘдёҚеҸҜеҸҳзҡ„refеҜ№иұЎпјҢе…¶currentеұһжҖ§иғҪеҲқе§ӢеҢ–дј йҖ’зҡ„еҸӮж•°гҖӮ
    8. uselmperativeMethods иҮӘе®ҡд№үдҪҝз”Ёrefж—¶е…¬ејҖз»ҷзҲ¶з»„д»¶зҡ„е®һдҫӢеҖјгҖӮ
    9. useMutationEffect жӣҙж–°е…„ејҹз»„д»¶д№ӢеүҚпјҢе®ғеңЁreactжү§иЎҢе…¶DOM жӣҙж”№зҡ„еҗҢдёҖйҳ¶ж®өеҗҢжӯҘи§ҰеҸ‘гҖӮ
    10. useLayoutEffect Dom ж”№еҸҳеҗҺеҗҢжӯҘи§ҰеҸ‘пјҢдҪҝз”Ёе®ғжқҘд»ҺDOMиҜ»еҸ–еёғеұҖе№¶еҗҢжӯҘйҮҚж–°жёІжҹ“

React Hook и§ЈеҶідәҶд»Җд№Ҳй—®йўҳ

  • React Hooksдё»иҰҒи§ЈеҶізҡ„жҳҜзҠ¶жҖҒе…ұдә«пјҢзҠ¶жҖҒе…ұдә«еҚ•жҢҮе…ұдә«зҠ¶жҖҒйҖ»иҫ‘еӨҚз”ЁпјҢе№¶дёҚжҳҜжҢҮж•°жҚ®д№Ӣй—ҙзҡ„е…ұдә«гҖӮжҲ‘д»¬зҹҘйҒ“еңЁReact Hooksд№ӢеүҚпјҢи§ЈеҶізҠ¶жҖҒйҖ»иҫ‘еӨҚз”Ёзҡ„й—®йўҳйҖҡеёёдҪҝз”Ёhigher-order components д»ҘеҸҠ render DOMиҜ»еҸ–еёғеұҖе№¶еҗҢжӯҘйҮҚж–°жёІжҹ“гҖӮ

  • ж—ўз„¶е·Із»ҸжңүдәҶиҝҷдёӨз§Қи§ЈеҶіж–№жЎҲпјҢдёәд»Җд№ҲиҝҳиҰҒеј•е…ҘReact Hookпјҹ еҜ№дәҺhigher-order components д»ҘеҸҠ render DOMпјҢ React Hook зҡ„дјҳеҠҝжҳҜд»Җд№Ҳпјҹ

    PS: Hook жңҖеӨ§зҡ„дјҳеҠҝе…¶е®һиҝҳжҳҜеҜ№дәҺзҠ¶жҖҒйҖ»иҫ‘зҡ„еӨҚз”ЁдҫҝжҚ·пјҢиҝҳжңүд»Јз Ғзҡ„з®ҖжҙҒд»ҘеҸҠеё®еҠ©еҮҪж•°з»„д»¶еўһејәеҠҹиғҪгҖӮ

  • еҖҹеҠ©е®ҳж–№demoд»Јз ҒжқҘзңӢдёӢ дҪҝз”Ёhooks

    import { useState } from 'React';

    жҲ‘д»¬еҸҜд»ҘзңӢеҲ°еңЁReact HooksдёӯпјҢclass example з»„д»¶еҸҳжҲҗдәҶеҮҪж•°ејҸз»„д»¶пјҢдҪҶжҳҜиҝҷдёӘеҮҪж•°ејҸз»„д»¶еҚҙжӢҘжңүзҡ„иҮӘе·ұзҡ„зҠ¶жҖҒпјҢеҗҢж—¶иҝҳеҸҜд»Ҙжӣҙж–°иҮӘиә«зҡ„зҠ¶жҖҒгҖӮиҝҷдёҖеҲҮйғҪиҰҒеҪ’еҠҹдәҺuseState зҡ„ Hook, useState дјҡиҝ”еӣһдёҖеҜ№еҖјпјҢеҪ“еүҚзҠ¶жҖҒе’ҢдёҖдёӘи®©дҪ еҸҜд»Ҙжӣҙж–°е®ғзҡ„еҮҪж•°пјҲдёҠйқўhookжңүиҜҙжҳҺпјүпјҢдҪ еҸҜд»ҘеңЁдәӢд»¶еҮҪж•°дёӯдҪҝз”ЁжҲ–е…¶д»–дёҖдәӣең°ж–№и°ғз”ЁиҜҘеҮҪж•°пјҢ class з»„д»¶зҡ„this.setState, дҪҶжҳҜе®ғдёҚдјҡжҠҠж–°зҡ„StateдёҺж—§зҡ„StateиҝӣиЎҢеҗҲе№¶гҖӮ

    React Hooks е…ЁеұҖз»ҙжҠӨдәҶдёҖдёӘworkInProgressHookеҸҳйҮҸпјҢжҜҸдёҖж¬Ўи°ғеҸ–Hook API йғҪдјҡе…Ҳи°ғз”ЁCreateWorkInProgressHooks еҮҪж•°гҖӮ

    function createWorkInProgressHooks() {
    if (workInProgressHook === null) {
        if (firstWorkInProgresshook === null) {
            currentHook = firstCurrentHook;
            if (currentHook === null) {
                workInProgressHook = createHook();
            } else {
                // жӢ·иҙқеҪ“еүҚhook
                workInProgressHook = cloneHook(currentHook);
            }
            firstWorkInProgressHook = workInProgressHook;
        } else {
            // е·Із»ҸеӯҳеңЁ work-in-progressпјҢзӣҙжҺҘеӨҚз”Ё
            currentHook = firstCurrentHook;
            workInProgressHook = firstWorkInProgressHook;
        }
    } else {
        if (workInProgresshook.next === null) {
            let hook;
            if (currentHook === null) {
                hook = createHook();
            } else {
                currentHook = currentHook.next;
                if (currentHook === null) {
                    hook = createHook();
                } else {
                    hook = cloneHook(currentHook);
                }
            }
            // иҝҪеҠ еҲ°еҲ—иЎЁ
            workInProgressHook = workInProgressHook.next = hook;
        } else {
            currentHook = currentHook !== null ? currentHook.next | null;
            workInProgressHook = workInProgressHook.next;
        }
    }
    return workInProgressHook;
}

    еҪ“жҲ‘д»¬дәҶи§ЈReact Hooksзҡ„з®ҖеҚ•еҺҹзҗҶпјҢеҫ—еҲ°Hooksзҡ„дёІиҒ”дёҚжҳҜдёҖдёӘж•°з»„пјҢдҪҶжҳҜжҳҜдёҖдёӘй“ҫејҸзҡ„ж•°жҚ®з»“жһ„пјҢд»Һж №иҠӮзӮ№жҳҜдёҖдёӘworkInProgressHookеҗ‘дёӢйҖҡиҝҮnextиҝӣиЎҢдёІиҒ”гҖӮиҝҷд№ҹе°ұжҳҜдёәд»Җд№Ҳhooks дёҚиғҪеөҢеҘ—дҪҝз”ЁпјҢдёҚиғҪеңЁжқЎд»¶еҲӨж–ӯдёӯдҪҝз”ЁпјҢдёҚиғҪеңЁеҫӘзҺҜдёӯдҪҝз”ЁпјҢеҗҰеҲҷдјҡз ҙеқҸй“ҫејҸз»“жһ„гҖӮ

hooksдёҺз”ҹе‘Ҫе‘Ёжңҹ

  • еҮҪж•°з»„д»¶зҡ„жң¬иҙЁжҳҜеҮҪж•°пјҢжІЎжңүstateзҡ„жҰӮеҝөзҡ„пјҢеӣ жӯӨдёҚеӯҳеңЁз”ҹе‘Ҫе‘ЁжңҹдёҖиҜҙпјҢд»…д»…жҳҜдёҖдёӘrenderзҡ„еҮҪж•°иҖҢе·ІгҖӮ
  • еҰӮеј•е…ҘHooksд№ӢеҗҺе°ұеҸҳеҫ—дёҚеҗҢдәҶпјҢе®ғиғҪи®©з»„д»¶еңЁдёҚдҪҝз”Ёclassзҡ„жғ…еҶөдёӢжӢҘжңүstateпјҢжүҖд»Ҙе°ұжңүдәҶз”ҹе‘Ҫе‘Ёжңҹзҡ„жҰӮеҝөпјҢжүҖи°“зҡ„з”ҹе‘Ҫе‘Ёжңҹе…¶е®һе°ұжҳҜuseStateгҖҒuseEffectдёҺuseLayoutEffect()гҖӮ
  • hooks з»„д»¶пјҲдҪҝз”ЁдәҶhooksпјүжңүз”ҹе‘Ҫе‘ЁжңҹпјҢиҖҢеҮҪж•°з»„д»¶пјҲжІЎжңүдҪҝз”ЁhooksпјүжІЎжңүз”ҹе‘Ҫе‘ЁжңҹгҖӮ

Daily Analysis еҠ е…ҘдәҶдёҖдәӣдёӘдәәжғіжі•пјҢ React Hooks зҡ„еҫҲеӨҡж–№жі•жІЎжңүе…·дҪ“д»Ӣз»ҚдёҺдҪҝз”ЁпјҢжҜҸдёҖдёӘHooksеҮҪж•°йғҪеӯҳеңЁж·ұеұӮзҡ„ж¶өд№үпјҢжҜ”еҰӮ useMemoгҖҒuseReducer еҗҺйқўж–Үз« дјҡеҚ•зӢ¬и®Іи§ЈгҖӮThanksпҪһ
